CIE三刺激值系数表说明
=================

三刺激值系数表(D65xCIE.xlsx)包含了CIE标准照明体D65下的XYZ三刺激值系数。

文件格式:
- 列: 400nm-830nm波长，以1nm为间隔
- 行: 三行分别对应XYZ三刺激值系数

使用方法:
1. 将文件放置在_internal目录下
2. 系统会自动加载该文件用于计算光谱的三刺激值

若您没有现成的三刺激值系数表，可以从以下位置获取:
- CIE官方网站: https://cie.co.at/
- 或参考相关文献中的数据

注意:
如果无法加载该文件，tris()函数将无法正常工作。
