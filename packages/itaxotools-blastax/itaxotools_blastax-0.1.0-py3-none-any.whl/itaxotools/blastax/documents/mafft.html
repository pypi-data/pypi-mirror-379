MAFFT is a multiple sequence alignment program by Katoh written in C and bash for Linux.

It is integrated as a Python-wrapped component, allowing it to run directly without a bash terminal. The original bash script has been replaced with a Python implementation, though very few options are currently supported.

Homepage: https://mafft.cbrc.jp/alignment/server/index.html
MAFFTpy: https://github.com/iTaxoTools/MAFFTpy

Katoh, K., Misawa, K., Kuma, K., & Miyata, T. (2002). MAFFT: a novel method for rapid multiple sequence alignment based on fast Fourier transform. Nucleic Acids Research, 30(14), 3059-3066.
