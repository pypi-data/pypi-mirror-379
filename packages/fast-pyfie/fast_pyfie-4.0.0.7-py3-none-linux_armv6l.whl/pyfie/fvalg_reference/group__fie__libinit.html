INT FVALGAPI fnFIE_setup ( VOID   ) 

FIEライブラリの初期化

FIEライブラリ全体にかかわる初期化を行います。 FIEライブラリを使用する前に一度この関数を呼んでください。

戻り値:
F_ERR_NONE 正常終了

INT FVALGAPI fnFIE_is_setuped ( VOID   ) 

FIEライブラリの初期化チェック

FIEライブラリ全体にかかわる初期化が行われているかチェックします。 fnFIE_setup() が実行済みの場合は TRUE を、そうでない場合は FALSE を返します。

戻り値:
TRUE 初期化実行済み
FALSE 未初期化

INT FVALGAPI fnFIE_set_num_threads ( INT  threads  ) 

内部スレッド数設定

各FIEライブラリでの内部スレッド生成数を指定します。 本設定はFIEライブラリ全体にかかわる設定です。

注意:
本設定値はスレッドごとに適用されます。 FIEライブラリがアプリケーションのメインスレッドとは異なる ワーカースレッド内で実行される場合、そのワーカースレッドごとに 本関数を実行して設定する必要があります。
引数:
[in] threads スレッド数
戻り値:
F_ERR_NONE 正常終了
F_ERR_UNSUPPORTED サポートされていません
