不具合修正

  • #569 fnFIE_region_decode() で 2値画像に対して水平方向画像範囲外のリージョンを描画したとき, 結果画像が正しくない不具合を修正.
  • #570 fnFIE_img_copy() で 浮動小数点数画像 から 整数画像 へのコピー処理のサチュレーション処理を修正.
  • #581 直線生成パラメータの定義値の誤りを修正. 直線生成パラメータは以下の関数にて使用されています.
    • fnFIE_cg_calc_line_npoint()
    • fnFIE_cg_calc_center_point_and_lineB()
    • fnFIE_cg_calc_cross_2lineB()
    • fnFIE_cg_calc_centerline_2npoint()
    • fnFIE_cg_calc_cross_4npoint()
    • fnFIE_cg_calc_rectangle_4npoint()
    • fnFIE_modelview_npointA()
    • fnFIE_modelview_npointB()

  • #584 fnFIE_fpm_feature_window_alloc() で サーチウインドウのパラメータチェックが間違っている不具合を修正.
  • #587 fnFIE_fpm_feature_alloc() で 画像より大きなフィルタサイズのとき, メモリアクセス違反を起こす不具合を修正.
