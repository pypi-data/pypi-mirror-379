1. FIE module Updates

不具合修正

  • 下記の画像ファイル読込み関数で、 fnFIE_img_root_import_alloc() により確保した画像オブジェクトに対して読込みを実行した場合 画像オブジェクトが適切な形式であっても常にエラー終了となっていた 不具合を修正.
    fnFIE_load_png(), fnFIE_load_jpeg(), fnFIE_load_tiff(), fnFIE_read_imgdata()
  • fnFIE_niblack_threshold() 及び fnFIE_sauvola_threshold() で、メモリ不足エラーにより終了する際 メモリリークが発生する場合がある 不具合を修正.
  • FPM(特徴点応用マッチング) の 通常モード かつ 高精度ポーズ推定有効 に設定したサーチの実行をマルチスレッド環境で行った際、実行するたびにサーチ結果が変わる場合がある 不具合を修正.
  • #594: fnFIE_region_includes_xy() で、指定点 y座標 の小数部が 0.5 であった場合に誤判定をする場合がある 不具合を修正.
