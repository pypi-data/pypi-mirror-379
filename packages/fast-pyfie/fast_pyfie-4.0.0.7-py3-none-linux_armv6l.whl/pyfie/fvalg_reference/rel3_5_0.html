機能追加および、変更

  • fnFIE_sauvola_threshold()
    • F_IMG_FLOAT型とF_IMG_DOUBLE型の場合にパラメータdRの制限を緩和.
  • fnFIE_kernel_filter()
    • 特定環境でカーネル値がSHORT型の最小値と最大値の範囲に収まっている場合に、F_IMG_S16型とF_IMG_US16型の処理を高速化.
      なお、高速化する環境では処理結果が極僅かに変化する可能性があります.

  • fnFIE_medianMxN()
    • 処理の高速化.
  • fnFIE_img_copy()
    • F_IMG_BIN型以外の同画像型間の処理を高速化.
  • fnFIE_fpm_matching(), fnFIE_fpm_matching_feature(), fnFIE_fpm_matching_image()
    • 通常モード(F_FPM_NORMAL_MODE),高速モード(F_FPM_SPEED_MODE)の処理の高速化.

不具合修正

  • #679, #695 fnFIE_fpm_matching_feature()
    • 特定条件のとき、実行毎に結果が変化してサーチ性能が低下する問題（通常モード（F_FPM_NORMAL_MODE）時のみ発生）を修正.

  • #681 fnFIE_img_mask(), fnFIE_img_mask2()
    • 特定の環境と画像サイズと画像型（F_IMG_RGBQUAD）で処理を行った場合に異常終了する問題を修正.
  • #685 fnFIE_ml_sample_add_region(), fnFIE_ml_predict_region_shape()
    • リージョンから抽出される特徴量の一部がスケール不変ではなく、期待される識別性能が得られない場合があった問題を修正.
  • #687 fnFIE_img_clear()
    • F_IMG_I64型の画像でマイナス値を設定できなかった問題を修正.
  • #688 fnFIE_ml_do_train_boost(), fnFIE_ml_do_train_dtree()
    • 特定環境でメモリリークが発生する問題を修正.
  • #692 fnFIE_geotrans_perspective()
    • 特定の変換を行うと意図しない画像が出力される問題を修正.
  • #693 fnFIE_dm_execute()
    • 縦横比が3倍を超える長方形データマトリックスが読めない問題を修正.
  • #694 fnFIE_geotrans_estimate_perspective_matrix(), fnFIE_geotrans_estimate_affine_matrix(), fnFIE_geotrans_estimate_similarity_matrix()
    • 特定の点列で計算不能エラーが発生する問題を修正.

  • #700 fnFIE_sauvola_threshold()
    • 浮動小数点画像（F_IMG_DOUBLE, F_IMG_FLOAT）の画素値が一定値の領域で黒い縞模様が出力される問題を修正.
  • fnFIE_fpm_feature_window_alloc()
    • 通常モード(F_FPM_NORMAL_MODE)で不要なメモリを確保していた問題を修正.
  • fnFIE_img_filter_template_parallel()
    • 指定した1ブロックのサイズが画像サイズと同じ際にエラーとなる問題を修正.
