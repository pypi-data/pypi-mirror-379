[２次元エッジ検出]

説明

細線エッジ用パラメータ構造体

細線エッジ用パラメータを指定するための構造体です。
このパラメータを使用する場合、出力結果の勾配方向が となるため、 F_FPM_MATCH::polarity には F_FPM_EITHER_POLARITY を指定する必要があります。
