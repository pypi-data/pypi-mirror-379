  • fnFIE_img_hsv_to_gray()
    • 画像型 F_IMG_S16, F_IMG_US16, F_IMG_FLOAT, F_IMG_DOUBLE, F_IMG_RGBQUAD, F_IMG_RGBTRIPLE に新規対応.

  • fnFIE_img_rotate_clockwise90()
    • 画像型 F_IMG_I64, F_IMG_RGBQUAD, F_IMG_RGBTRIPLE に新規対応.

  • fnFIE_img_min()
    • 画像型 F_IMG_I32, F_IMG_UI32, F_IMG_I64, F_IMG_FLOAT, F_IMG_RGBQUAD, F_IMG_RGBTRIPLE に新規対応.

  • fnFIE_gs2_search()
    • 特定の条件下における処理の高速化.

  • fnFIE_bilateral(), fnFIE_geotrans_affine(), fnFIE_geotrans_perspective(), fnFIE_geotrans_warpping()
    • 特定の条件下における処理の高速化.

  • fnFIE_fpm_matching(), fnFIE_fpm_matching_feature()
    • 処理を高速化.

不具合修正

  • #663 fnFIE_fit_circle2()
    • 特定点群を入力した場合に標準エラー出力へエラーメッセージが出力されていた問題を修正.

  • #759 fnFIE_mat_linsolve2()
    • マルチスレッド環境下で稀にフリーズする問題を修正.

  • fnFIE_niblack_threshold(), fnFIE_sauvola_threshold()
    • 局所領域のサイズが大きく、特定の画像の場合に正常に計算できなくなる可能性があったため修正.

  • fnFIE_cg_enclosing_circle_calc()
    • 特定の点群を入力した場合に正常に計算できない可能性があったため修正.
