構造体 DLINE3_T

説明

３次元空間内の直線（ベクトル方程式）

直線上の１点Aの位置ベクトル と方向ベクトル により， 直線上の任意の点Pの位置ベクトル を （ t は実数）と表します． このとき，方向ベクトルを単位ベクトルにとることにより，パラメータ t (の絶対値)は点Aからの距離となります．
