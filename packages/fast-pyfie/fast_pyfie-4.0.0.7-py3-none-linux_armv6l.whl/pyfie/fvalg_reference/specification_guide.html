ソフトウェアプロテクトについて
FIE ライブラリの各関数にはソフトウェアプロテクトが施されており、 必用なライセンス情報がコンピューター上に存在していない場合には 関数はライセンスエラーとなり異常終了します。このとき、 返値がINT型である関数では F_ERR_NO_LICENCE エラーが、 返値がポインタ型である関数では NULL が返値として返されます。
インプレース/ハーフインプレース処理について
FIE ライブラリでは、特別な記述のない限りインプレース又はハーフインプレース 処理を行うことは出来ません。インプレース又はハーフインプレース処理とは、 入力データ（画像等）において、入力と出力が同じメモリ領域となる処理を言います。 これらについては特に引数チェックを行われないため、 もし処理が行われた場合の処理結果は不定となります。
