ボロノイ図 辺情報構造体

ボロノイ図の各辺に対する情報を保持する構造体です。 ボロノイ辺は、 ax+by+c=0 式の係数a, b, c にて表します。

enum f_cg_vrni_egde_type F_CG_VE_INFO::type

ボロノイ辺の種類 以下のうちいずれか。

  • F_CG_SEGMENT_TYPE ：線分であるボロノイ辺
  • F_CG_ST_PNT_RAY ：始点を持つ半直線であるボロノイ辺
  • F_CG_ED_PNT_RAY ：終点を持つ半直線であるボロノイ辺
