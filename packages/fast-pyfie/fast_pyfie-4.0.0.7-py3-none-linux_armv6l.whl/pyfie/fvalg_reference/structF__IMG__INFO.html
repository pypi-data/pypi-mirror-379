構造体 F_IMG_INFO
[画像ファイルアクセス]

説明

FIEイメージ用ヘッダー構造体

低レベル画像ファイルアクセス関数群で使用される、 画像種別取得用構造体です。

参照:
fnFIE_check_png_info(), fnFIE_check_jpeg_info(), fnFIE_check_tiff_info()

画像の高さ

UINT F_IMG_INFO::type

画像の画素種別
下記の何れかの値が格納されます。

  • F_IMG_BIN
  • F_IMG_UC8
  • F_IMG_S16
  • F_IMG_US16
  • F_IMG_DOUBLE
  • F_IMG_RGBQUAD
