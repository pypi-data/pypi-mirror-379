バブルソート

バブルソートアルゴリズムにより、配列を並べ替えます。 比較関数 comp は、２つの要素間の関係を示す関数で、ユーザーが指定します。 この比較関数 comp は次の型で定義します。

INT (FVALGAPI *comp)(const VOID *elem1, const VOID *elem2);

返値は２つの要素の大小関係を表す値を返します。

  • (返値)<0 : elem1 は elem2 より小さい
  • (返値)==0 : elem1 は elem2 と等しい
  • (返値)>0 : elem1 は elem2 より大きい
本関数は、この関係に従って、配列を昇順に並び替えます。 配列を降順で並び替えたい場合は、比較関数 comp の大小関係を反転させて実行します。

なお高速化のため、パラメータのチェックをしていません。
