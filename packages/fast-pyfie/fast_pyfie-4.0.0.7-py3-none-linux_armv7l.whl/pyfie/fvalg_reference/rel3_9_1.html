1. FIE module Updates

機能追加および、変更

  • fnFIE_img_minmaxdens()
    • maxdens, mindensのうち不要なものをNULLに設定できるよう変更

  • データマトリックス読み取り
    • 長方形データマトリックスに対応、あわせてドキュメントにあった長方形データマトリックス非推奨の文言を削除

不具合修正

  • fnFIE_projection()
    • メモリエラー時に正常終了しない場合がある不具合を修正

  • fnFIE_qr_execute()
    • メモリエラー時に正常終了しない場合がある不具合を修正

  • fnFIE_fit_polynomial()
    • メモリエラー時に正常終了しない場合がある不具合を修正

  • fnFIE_fpm_matching_feature()
    • メモリエラー時に正常終了しない場合がある不具合を修正

サンプルコード追加

  • サーチ対象画像にマスクを利用し、適切な範囲でFPMを行う
    • fnFIE_fpm_feature_mask()

  • fnFIE_edge_binary_boundaryとブロブ解析を用いた高速サーチ
    • fnFIE_edge_binary_boundary()

  • 指定した点を囲む輪郭を元にリージョンを生成

ドキュメント修正

  • 照度差ステレオ 説明
    • 写真の差し替え（硬貨→卵の殻）
    • 図中の補助関数名の誤りを修正

  • fnFIE_save_bmp()
    • 引数の説明の誤りを修正

  • 幾何変換 説明
    • リンク先の誤りを修正

  • fnFIE_vectarray_pop_back()
    • 引数の名前の誤りを修正

  • fnFIE_img_child_free()
    • 不要な関数ドキュメントが表示されていたため削除。今後のチャイルド画像の開放にはfnFIE_free_object()を使用すること

  • ドキュメント FIEライブラリの使い方(FVL/LNXユーザー向け)
    • LNXにおいてカメラキャリブレーションライブラリの販売はしていないため、関連する記述を削除
