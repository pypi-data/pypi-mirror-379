FIE関数の追加

fnFIE_から始まる画像処理関数群はFIEオブジェクトを利用する新しい処理関数群として追加されました。
FIEオブジェクトとは、画像、リージョン、StructureElement等のオブジェクト(データのまとまり) をC言語にて適切に扱えるよう設けられた、 オブジェクトインスタンスハンドラです。
オブジェクトインスタンスをFHANDLE型にて扱うことによって、データのカプセル化・多様性を クラスの無いC言語で得ることができます。
