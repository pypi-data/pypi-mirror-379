機能追加および、変更

FPM(特徴点応用マッチング)

既存関数の特徴量抽出手法に強度上限付ソーベル、強度上限付相関エッジの2種類を追加.
  • データ構造 : 特徴量計測手法
  • データ構造 : 特徴量計測用共用体
  • 関数 : FPM オブジェクト生成
  • 関数 : 特徴量パラメータ取得
  • 関数 : FPM画像オブジェクトからのマッチング実行
  • 関数 : 一連のマッチング実行
  • 関数 : 特徴量を取得
  • 関数 : ウィンドウを指定して特徴量を取得
  • 関数 : 特徴量抽出手法の取得

マルチコアCPUでの処理速度改善.
  • 関数 : FPM オブジェクト生成
  • 関数 : FPM画像オブジェクトからのマッチング実行
  • 関数 : FPM特徴量オブジェクトからのマッチング実行
  • 関数 : 一連のマッチング実行
  • 関数 : 特徴量を取得
  • 関数 : ウィンドウを指定して特徴量を取得

不具合修正

  • #533: fnFIE_geotrans_affine() で 出力画像が全て領域外になる場合、領域外画素をクリアしないオプションを指定していてもクリアされてしまうバグを修正
  • #534: fnFIE_img_copy() で 符号付型の画像から 2 値画像へコピーする時 負の値をもつ画素が 黒ではなく白になってしまうバグを修正.
  • #539: fnFIE_calib_calc_1shot() で 光軸中心が画像右側にあると正常にカメラキャリブレーションできないバグを修正.
  • #541: fnFIE_sauvola_threshold() および fnFIE_niblack_threshold() で 入力画像型が US16 の時 2値化結果の白黒が反転する場合があるバグを修正
