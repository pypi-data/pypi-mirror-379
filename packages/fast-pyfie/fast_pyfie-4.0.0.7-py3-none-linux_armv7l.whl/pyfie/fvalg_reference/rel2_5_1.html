1. FIE module Updates

機能追加及び仕様変更

  • Version 2.4.4にて、FVL/LNX 環境でのファイルアクセス関数で、フロッピーディスクドライブの代わりにCF内のフォルダをマウントするように 修正を行ったが、CDドライブもマウントされなくなってしまったため、CDドライブはマウントするように修正.
  • FVL/LNX 環境でのフルパス取得時にディレクトリが階層になっていてもパスを取得出来るように変更.
