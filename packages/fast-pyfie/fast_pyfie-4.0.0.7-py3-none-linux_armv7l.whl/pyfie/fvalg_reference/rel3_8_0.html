不具合修正

  • #753 fnFIE_fpm_alloc(), fnFIE_fpm_feature_alloc(), fnFIE_fpm_feature_window_alloc()
    • 特定のパラメータにNULLを入力するとNULLポインタデリファレンスが起きる問題を修正.
  • #754 fnFIE_img_add()
    • 入力画像1の画像型を F_IMG_UC8、入力画像2の画像型を F_IMG_S16、出力画像の画像型を F_IMG_US16 にすると計算結果が不正になる問題を修正.
  • #757 fnFIE_qr_execute()
    • バージョン28（129x129セル）以上のQRコードを読み取ろうとすると読み取り失敗しメモリ不足エラーが発生する問題を修正.
  • #760 fnFIE_save_tiff(), fnFIE_save_core_tiff()
    • 圧縮レベル (quality) のパラメータが反映されない問題を修正.
  • #761 fnFIE_load_tiff(), fnFIE_load_core_tiff(), fnFIE_check_tiff_info()
    • 圧縮形式を F_TIFF_COMPRESSION_JPEG と設定した fnFIE_save_tiff() により保存したカラー画像の読み込みに失敗する問題を修正.
  • FVALG OSS モジュールが依存するサードパーティ製オープンソースソフトウェア (OSS) の一部をより新しいバージョンに更新. これによりいくつかの脆弱性が修正された.

ドキュメント修正

  • fnFIE_save_png()
    • カラータイプの誤りを修正し、圧縮レベルの説明を改善.
  • #758 fnFIE_mat_linsolve(), fnFIE_mat_linsolve2()
    • ゼロ判定閾値のパラメータの入出力属性の誤りを修正.
