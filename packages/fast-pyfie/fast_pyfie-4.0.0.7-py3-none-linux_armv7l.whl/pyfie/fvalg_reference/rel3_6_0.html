FIE関数の追加

画像フィルタ：フィルタ全般

  • ウィーナフィルタによる画像復元 / fnFIE_wiener()
  • Inf(無限大)を数値に置き換える / fnFIE_replace_inf()
  • NaN( 非数 )を数値に置き換える / fnFIE_replace_nan()

画像フィルタ：高速フーリエ変換

  • splitted format の複素数画像間の乗算 / fnFIE_fft2_mul_DDtoDD()
  • splitted format の複素数画像間の除算 / fnFIE_fft2_div_DDtoDD()

機能追加および、変更

  • fnFIE_fpm_feature_mask(), fnFIE_fpm_matching_image(), fnFIE_fpm_matching_feature(), fnFIE_fpm_matching()
    • 特定環境でのFPM高速化

  • fnFIE_edge_corr(), fnFIE_edge_corr_subpix()
    • 処理を高速化
    • 画像型 F_IMG_S16, F_IMG_US16, F_IMG_DOUBLE に新規対応

  • #696 #715 fnFIE_make_histogram()
    • ヒストグラム生成の処理速度向上

  • fnFIE_img_child_attach(), fnFIE_img_child_attach_single_ch(), fnFIE_img_child_detach()
    • 特定環境においてチャイルド画像割り当てを高速化

  • fnFIE_medianMxN()
    • 画像型が F_IMG_DOUBLE かつフィルタの幅と高さがともに3の場合において処理を高速化

不具合修正

  • #638 fnFIE_save_bmp()
    • 24bit カラー ビットマップ画像保存時、ヘッダ情報にあった不具合を修正
  • #708 fnFIE_fpm_matching_image(), fnFIE_fpm_matching_feature(), fnFIE_fpm_matching()
    • 高速モードのサーチ中、特徴点が極めて少ない場合にメモリ不足エラーが発生することがあった問題を修正

  • #709 fnFIE_edge_sobel(), fnFIE_edge_sobel_subpix(), fnFIE_edge_corr(), fnFIE_edge_corr_subpix()
    • 検出エッジ数0でも強度画像を出力するように修正　従来は元の画像を保持していた
  • #710 fnFIE_make_pseudo_colormap(), fnFIE_make_pseudo_colormap_dbl()
    • 一部の虹色疑似カラーマップ生成で範囲外メモリアクセスを解消
    • 一部の虹色疑似カラーマップ生成で結果が変化することがあった不具合を修正
  • #711 fnFIE_mat_chol()
    • 不正な引数を入力した場合にエラーメッセージが表示されることがあった不具合を修正
  • #712 fnFIE_gs2_pattern_alloc(), fnFIE_gs2_search(), fnFIE_gs2_search_enforce(), fnFIE_gs2_search_enforce2(), fnFIE_img_shrink()
    • 一部環境で他環境と異なる画像ピラミッドや圧縮画像を生成していた不具合を修正

ドキュメント修正

  • #589 無名だった列挙型を命名（ボーダー処理モード、sobelの計算モード、線(曲線)抽出フィルタの抽出する線の種類）
  • #621 円近似 fnFIE_fit_circle2() などのドキュメント誤りを修正 (入力点群の個数制限等)
  • #699 ２値ブローブ解析の特徴量一覧表で、単色解析モードで計算可能な特徴量の一部が計算不可と表記されていたので修正
  • fnFIE_img_div() fnFIE_img_div_const2() ドキュメントの引数にあった誤字を修正
  • fnFIE_draw_rect() 回転角の単位がラジアンであることを明記
  • fnFIE_modelview_set_points() 点群が同一直線状に並んだ際の挙動を明記
  • fnFIE_edge1d_line2() などにおいて、エッジ取得間隔はエッジ方向が等しいエッジ同士にのみ適用されることを明記
  • ドキュメントグループ 「画像フィルタ：FFT変換」の表記を「画像フィルタ：高速フーリエ変換」に変更
  • 高速フーリエ変換のドキュメントにて interleaved format 形式の画像を fnFIE_packing_ch() により作成できることを追記
  • 高速フーリエ変換のサンプルコードを追加
  • 相関エッジフィルタのドキュメントを改善
  • FPM高精度ポーズ推定のドキュメントを改善
  • 画像オブジェクトの誤字修正、関数リンク追加
  • サンプルコード一覧の修正
  • FIE for Linux のための使用法を追加
