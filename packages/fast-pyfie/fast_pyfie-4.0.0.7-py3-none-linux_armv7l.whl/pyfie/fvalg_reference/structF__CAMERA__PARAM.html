カメラ内部パラメータ構造体 (intrinsic parameters)

カメラ内部パラメータ行列

の成分と、放射歪み係数 k1, k2 からなる構造体です。


変数

DOUBLE a
 カメラ内部パラメータ alpha (スケール係数 水平方向)

 カメラ内部パラメータ beta (スケール係数 垂直方向)
DOUBLE g
 カメラ内部パラメータ gamma (スキュー)
DOUBLE u
 カメラ内部パラメータ u0 (光軸中心 水平方向)
DOUBLE v
 カメラ内部パラメータ v0 (光軸中心 垂直方向)
DOUBLE k1
 放射歪み係数 k1 (光軸中心からのL2ノルムの2乗に掛ける係数)
DOUBLE k2
 放射歪み係数 k2 (光軸中心からのL2ノルムの4乗に掛ける係数)
