構造体 F_EDGE_CLUST
[２次元エッジ操作]

説明

エッジ点クラスタ構造体

  • F_EDGE_CLUST ひとつでひとつの曲線を表しています。
  • pindex に保持されているデータはエッジ点の座標ではなく、エッジ点群配列のインデックスとなります。
  • pindex[0] と pindex[num-1] が同一の場合閉曲線を表し、そうでない場合開曲線を表します。
参照:
fnFIE_edge_connecting() , fnFIE_edge_connecting2() , fnFIE_refilter_edge_by_clust()
