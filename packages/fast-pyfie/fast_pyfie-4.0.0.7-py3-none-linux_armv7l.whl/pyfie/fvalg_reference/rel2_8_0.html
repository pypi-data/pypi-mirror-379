1. FIE module Updates

FIE関数の追加

算術演算

  • 画像間逆正接演算
  • 画像指数関数値演算
  • 画像対数関数値演算

フィルタ全般

  • 適応的平滑化フィルタ (3×3)
  • 適応的平滑化フィルタ (m×n)
  • ガボール フィルタ

リージョン処理

  • リージョンが指定点を含むかの判定
  • リージョン同士が重なりを持つかの判定

幾何計算

  • 楕円と直線の交点の算出
  • 点群からのn次多項式近似

機能追加および、変更

  • fnFIE_img_mirror()
    • 入出力画像の対応画像型式に 単精度浮動小数点数画像(F_IMG_FLOAT) を追加.
  • fnFIE_copy_border()
    • 入出力画像の対応画像型式に 単精度浮動小数点数画像(F_IMG_FLOAT) を追加.
  • fnFIE_img_copy()
    • 入出力画像の対応画像型式に 符号付き32bit濃淡画像 (F_IMG_I32) を追加.
    • 入出力画像の対応画像型式に 32bit濃淡画像 (F_IMG_UI32) を追加.
    • 入出力画像の対応画像型式に 符号付き64bit濃淡画像 (F_IMG_I64) を追加.
    • 入出力画像の対応画像型式に 24bitパッキングカラー (F_IMG_RGBTRIPLE) を追加.
  • fnFIE_varianceMxN()
    • 処理の高速化.
    • 入出力画像に対するチャネル数の制限を解除.
    • フィルタサイズの制限を解除.
  • fnFIE_averageMxN()
    • 処理の高速化.
    • 入出力画像に対するチャネル数の制限を解除.
    • フィルタサイズの制限を解除.
  • fnFIE_binarize()
    • 入出力画像のチャネル数制限を解除.
  • fnFIE_unpacking_rgb()
    • 入力画像の対応画像型式に 24bitパッキングカラー画像 (F_IMG_RGBTRIPLE) を追加.
    • チャネル数が 4 である出力画像への対応を追加.
  • fnFIE_packing_rgb()
    • 出力画像の対応画像型式に 24bitパッキングカラー画像 (F_IMG_RGBTRIPLE) を追加.
    • チャネル数が 4 である入力画像への対応を追加.

不具合修正

  • #555 fnFIE_cg_calc_cross_circle_and_lineB() で 直線が垂直に近い場合 2 つの交点を 1 つと誤回答していたバグを修正.
  • #561 fnFIE_edge1d_line2() で 64bit 環境における実行時にメモリアクセス違反を起こす場合があったバグを修正.
  • #564 fnFIE_fpm_matching() で 高速モードによるサーチ実行時にメモリアクセス違反を起こす場合があったバグを修正.
