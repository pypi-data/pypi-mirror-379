関数インターフェイスの変更

#500 での変更に伴い、画像メモリを扱う下記関数の引数型が変更されています。
32bit バイナリ環境ではこの変更によるユーザープログラムの変更は必要有りませんが、 コンパイラの設定によってはワーニングが新たに出るようになる可能性が有ります。ご注意ください。

  • 旧
    1. FHANDLE FVALGAPI fnFIE_img_root_import_alloc( VOID **adrss, INT channels, INT type, INT step, INT width, INT height );
    2. INT FVALGAPI fnFIE_img_get_params( FHANDLE himg, INT *channels, INT *type, INT *step, INT *width, INT *height );
    3. INT FVALGAPI fnFIE_img_get_step( FHANDLE himg );
  • 新
    1. FHANDLE FVALGAPI fnFIE_img_root_import_alloc( VOID **adrss, INT channels, INT type, INT_PTR step, INT width, INT height );
    2. INT FVALGAPI fnFIE_img_get_params( FHANDLE himg, INT *channels, INT *type, INT_PTR *step, INT *width, INT *height );
    3. INT_PTR FVALGAPI fnFIE_img_get_step( FHANDLE himg );
