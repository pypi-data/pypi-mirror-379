1. FIE module Updates

機能追加および、変更

  • fnFIE_fit_circle()
    • 近似モード F_FIT_MESTIMATOR と F_FIT_MESTIMATOR2 の処理を高速化.

不具合修正

  • fnFIE_img_copy()
    • 特定の環境において、32bitパッキングカラー画像(F_IMG_RGBQUAD)の画像コピーが顕著に遅くなる問題を修正.
  • fnFIE_vect_div_elem_vv()
    • 特定の環境において浮動小数点ゼロ同士の除算でx87コントロールレジスタが変更されてしまうため、同環境でのゼロ除算はDBL_MINの除算になるように変更.
  • fnFIE_img_mul()
    • 16bit濃淡画像(F_IMG_US16)の画像乗算で最大値同士を処理すると乗算オーバーフローが発生し結果がおかしくなる問題を修正.
  • fnFIE_img_mask_mul()
    • 16bit濃淡画像(F_IMG_US16)の画像乗算で最大値同士を処理すると乗算オーバーフローが発生し結果がおかしくなる問題を修正.
  • #610:fnFIE_img_maxpixel()
    • 全ての画素値が負である 倍精度浮動小数点画像(F_IMG_DOUBLE)に対して間違った処理結果を返すバグを修正.
  • #606:fnFIE_measure_get_list()
    • 下記の特徴量でフィルタを掛けると、結果がおかしくなるバグの修正.
      F_FEATURE_DCMIN, F_FEATURE_DCMAX, F_FEATURE_DSMIN, F_FEATURE_DSMAX, F_FEATURE_M30, F_FEATURE_M21
