1. FIE module Updates

FIE関数の追加

機械学習

  • ナイーブベイズ分類器による機械学習
  • ナイーブベイズ分類器による学習パラメータを取得する
  • ナイーブベイズ分類器による、出力クラスの生起確率の予測
  • ナイーブベイズ分類器の学習で使用されたクラスラベルの一覧を取得する
  • ナイーブベイズ分類器による、出力クラスの事前確率の取得
  • ナイーブベイズ分類器で学習された尤度モデルのパラメータを取得する

フィルタ全般

  • Mean-Shift 法による平滑化フィルタ

ビジョンツール

  • ピーク探索

幾何変換

  • 相似変換の変換行列を計算
  • ユークリッド変換の変換行列を計算

サンプルコード追加

  • 応用サンプルコード
    • 一次元エッジ検出(矩形)
    • 画像のPack/Unpack/チャネル統合/結合/分割

機能追加および、変更

  • fnFIE_img_copy()
    • 2値画像から2値画像以外のすべての型への画像コピーを高速化.

不具合修正

  • 画像オブジェクト
    • 複数スレッドから同じ画像に対してチャイルド画像をアタッチデタッチ可能になるように修正.
      以前は同操作によってメモリアクセス違反やメモリリークを引き起こす危険性があった.
  • #630 fnFIE_fit_circle2()
    • 誤った誤差計算式が使用される問題を修正.
  • #631 fnFIE_bfilter()
    • 出力画像をチャイルド画像で指定した際に結果が異なるものになるバグを修正.
  • #635, 636 FPMオブジェクト
    • マスクをかけたFPMオブジェクトの複製時にアクセス違反が発生する問題を修正(#635).
    • マスクをかけたFPMオブジェクトのコピーに失敗する問題を修正(#636).
  • #639, 640 fnFIE_band_threshold()
    • S16 画像型で値域付近の閾値を指定した場合, 期待する処理結果を得られない問題を修正(#639).
    • 閾値の指定範囲についての挙動が仕様と異なっている問題を修正(#640).
  • fnFIE_ml_predict()
    • Real adaBoostでカテゴリ属性の識別を行う際に、入力ベクトルの値の小数点以下が切り捨てられない問題を修正.
  • #609 fnFIE_img_filter_template_1x1()
  • #637 fnFIE_img_shrink()
    • サンプルコード修正.

ドキュメント修正

  • fnFIE_create_box_se()
  • fnFIE_create_disk_se()
  • fnFIE_create_line_se()
    • 構造要素作成関数の解放処理について追記.
  • #619 fnFIE_texture_first_order_statistics()
  • #633 fnFIE_texture_coocurrence_matrix()
  • #620 色彩検査　説明
  • #642 ２次元エッジ検出 マクロ定義
  • fnFIE_edge_corr()
  • fnFIE_edge_corr_subpix()
    • ドキュメントの誤りを修正.
  • 遺伝的アルゴリズム　型定義
  • フィルターテンプレート　型定義
    • 一覧表示が崩れていた問題を修正.
