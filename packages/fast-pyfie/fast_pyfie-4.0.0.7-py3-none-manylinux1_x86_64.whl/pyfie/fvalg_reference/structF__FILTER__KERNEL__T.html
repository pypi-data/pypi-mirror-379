説明

フィルタカーネル構造体

fnFIE_kernel_filter() 等で使用されるフィルタカーネル指定用構造体です。 カーネルの値は、可変長１次元配列 p により定義します。 p には、size_x * size_y * sizeof(DOUBLE) の領域が必要です。 カーネルの各値は、カーネルの値を K(x,y) と表記すれば、

K(x,y) = p[ (y+anchor_y) * size_x + (x+anchor_x) ]
但し
-anchor_x ≦ x ≦ size_x-anchor_x-1
-anchor_y ≦ y ≦ size_y-anchor_y-1

と定義されます。
