構造体 LINE

説明

直線指定用構造体(INT)

直線を の方程式で表すときの、 パラメータ a, b, c をスケールされた整数で表す構造体です。 パラメータ a, b は 65536倍の整数、パラメータ c は16倍の整数で表します。 a, b のスケール値と c のスケール値が違っていることに注意してください。


変数

INT a
INT b
INT c

構造体

INT LINE::a

a * 65536 (32 bit)

INT LINE::b

b * 65536 (32 bit)

INT LINE::c

c * 16 (32 bit)
