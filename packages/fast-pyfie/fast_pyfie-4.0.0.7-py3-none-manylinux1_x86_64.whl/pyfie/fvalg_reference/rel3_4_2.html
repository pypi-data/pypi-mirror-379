1. FIE module Updates

機能追加および、変更

  • fnFIE_copy_border()
    • 二値画像におけるF_BORDER_CONTINUOUSボーダー拡張を高速化
  • fnFIE_fpm_matching(), fnFIE_fpm_matching_image(), fnFIE_fpm_matching_feature()
    • 内部処理の細かな修正により、わずかに高速化
  • fnFIE_img_child_attach()
    • 同じルート画像に割り当てたチャイルド画像に再割り当てする際の処理を高速化
  • fnFIE_img_get_ch_adrs(), fnFIE_img_get_params()
    • チャイルド画像に対する処理を高速化
  • #626 fnFIE_erosion_se(), fnFIE_dilation_se() と、これらに関連するモルフォロジ演算
    • 高速化のため、ボーダー拡張の挙動を修正
  • 本ライブラリをWindowsリモートデスクトップ経由で利用できるように変更。従来はリモートデスクトップ経由で使用しようとすると、ライセンスエラーとなっていた。

不具合修正

  • fnFIE_projection()
    • 二値画像を処理する際、不要でも画像の終端より32bit先まで読み込んでしまう問題を修正
  • #591 fnFIE_fpm_get_relative_edge()
    • コピーしたFPMオブジェクトでは動作していなかった問題を修正
  • #677 fnFIE_erosion(), fnFIE_dilation() と、これらに関連するモルフォロジ演算
    • バージョン 3.4.0 でF_IMG_UC8型を除き処理速度が遅くなった問題（特にF_IMG_BIN型で顕著）を修正

  • #678 fnFIE_median(), fnFIE_rank_filter()
    • 画像の幅または高さが3未満のときに、画像の領域外を参照する問題を修正
  • fnFIE_geotrans_estimate_perspective_matrix()
    • 特定の環境で処理が無限ループとなる問題を修正

ドキュメント修正

  • モルフォロジ概要説明
    • boundary condition の説明を修正
  • fnFIE_erosion_se(), fnFIE_dilation_se()
    • 境界部分処理の説明を修正
