説明

３次元空間内の平面指定構造体 (FLOAT)

３次元空間内の平面を、平面内のある１点と パラメータp により表す構造体です。

４次のパラメータp は下式を表します。
p[0] * x + p[1] * y + p[2] * z + p[3] = 0, norm(p[0..2])=1
