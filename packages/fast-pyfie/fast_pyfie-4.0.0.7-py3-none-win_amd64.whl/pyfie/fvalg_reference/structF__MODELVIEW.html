構造体 F_MODELVIEW
[座標系変換行列生成]

説明

座標系変換行列生成用の構造体

座標系変換行列生成をするための構造体です。 本構造体を利用する際は、必ず fnFIE_modelview_alloc_param() 関数を利用して生成してください。

構造体

FMATRIX* F_MODELVIEW::forward

スクリーン座標からワールド座標に変換するときに使用する同次行列（３×３行列）

FMATRIX* F_MODELVIEW::backward

ワールド座標からスクリーン座標に変換するときに使用する同次行列（３×３行列）

INT F_MODELVIEW::reverse_y

ワールド座標のＹ軸方向設定

  • TRUE ：Ｙ軸反転有り(Ｙ軸正方向上向き)
  • FALSE ：Ｙ軸反転無し(Ｙ軸正方向下向き)

DOUBLE F_MODELVIEW::angle

スクリーン座標から見たときの、ワールド座標への回転角（単位：ラジアン）

DOUBLE F_MODELVIEW::move_x

スクリーン座標から見たときの、ワールド座標へのＸ方向移動量

DOUBLE F_MODELVIEW::move_y

スクリーン座標から見たときの、ワールド座標へのＹ方向移動量

DOUBLE F_MODELVIEW::scale_x

スクリーン座標から見たときの、ワールド座標のＸ方向倍率

DOUBLE F_MODELVIEW::scale_y

スクリーン座標から見たときの、ワールド座標のＹ方向倍率
