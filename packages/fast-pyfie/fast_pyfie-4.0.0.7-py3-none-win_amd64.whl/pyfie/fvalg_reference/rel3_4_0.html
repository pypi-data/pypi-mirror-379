機能追加および、変更

  • fnFIE_polar_trans_img()
  • fnFIE_polar_trans_img_inv()
  • fnFIE_polar_trans_map()

  • fnFIE_polar_trans_pnt_inv()

  • #571 fnFIE_add_noise_gauss(),fnFIE_add_noise_salt_and_pepper()
    • 8bit濃淡画像(F_IMG_UC8),符号付き16bit濃淡画像(F_IMG_S16),16bit濃淡画像(F_IMG_US16)でのメモリ使用量を改善.
  • #571 fnFIE_add_noise_salt_and_pepper()
    • 2値画像(F_IMG_BIN),8bit濃淡画像(F_IMG_UC8),符号付き16bit濃淡画像(F_IMG_S16),16bit濃淡画像(F_IMG_US16)でのメモリ使用量を改善.

    • 2値画像(F_IMG_BIN)の処理を高速化.
  • #665 fnFIE_save_bmp()
    • 2値画像(F_IMG_BIN)、3チャネル8bit濃淡画像(F_IMG_UC8)、32bitパッキングカラー(F_IMG_RGBQUAD)の処理を高速化.

不具合修正

  • fnFIE_fit_circle2()
    • 最小メディアン法(F_FIT_LMEDS, F_FIT_LMEDS2)で、ランダムサンプリング点群が一直線上に並んだ際にエラーとなる問題を修正.
  • #660 fnFIE_edge1d_projection_line()
    • 計測ライン長が1となる線分を入力した際に、0割りが発生する問題を修正.
  • #626, #628 モルフォロジ演算全般
    • 特定環境で画像端の処理結果が異なる問題を修正.
  • #653 #654 fnFIE_fpm_matching_feature()
    • 特定の点群で通常モード(F_FPM_NORMAL_MODE)のサーチを行なうとヒープ破壊が起きる問題を修正(#653).
    • ※2018/11/2追記
      特定の環境・点群で高速モード(F_FPM_SPEED_MODE)のサーチを行った場合も上記と同様のヒープ破壊が起きていた問題を修正(#653).
    • 通常モード(F_FPM_NORMAL_MODE)と点情報によるFPMオブジェクトを用いたサーチで、画像端の対象が正常にサーチできない問題を修正(#654).
  • #655 fnFIE_fpm_matching()
    • サーチ対象画像よりパタン画像が大きい場合にメモリ不足エラー(F_ERR_NOMEMORY)となる問題を修正.
  • #657 fnFIE_polar_trans_img(),fnFIE_polar_trans_map(),fnFIE_polar_trans_pnt_inv()
    • 特定の回転軸位置で処理を行なった場合に画像端の一行が正確に求められない問題を修正.
  • #532 #662 #664 fnFIE_geotrans_affine()
    • Lanczos法補間（F_SAMPLING_LANCZOS）で画像周辺が正常に処理されない問題を修正(#532).
    • 出力画像の有効領域とマスク画像の有効領域が数画素ずれる問題を修正(#662).
    • 倍精度浮動小数点画像(F_IMG_DOUBLE)でLanczos法補間(F_SAMPLING_LANCZOS)計算が異常となる問題を修正(#664).
  • #662 fnFIE_geotrans_perspective()
    • 特定の行列が入力された際に正常な出力が得られない問題を修正.
    • 出力画像の有効領域とマスク画像の有効領域が数画素ずれる問題を修正.
  • #666 fnFIE_color_convert()
    • 特定環境でHSV(FAST独自仕様：F_CS_HSV_FAST)の倍精度浮動小数点画像(F_IMG_DOUBLE)から8bit濃淡画像(F_IMG_UC8)への変換が正常に行われない問題を修正.

  • 色空間変換.
    • HSV濃淡化の注意書き追加.
  • #626 モルフォロジ演算.
