不具合修正

  • #1047(771)　fnFIE_fpm_matching_feature()
    • 高速モードで発見できるが通常モードで発見できない問題を修正.　
      • この変更により通常モードでは画像によっては「粗い精度でのマッチングでの低圧縮スコア」が50以上増加します。そのため、これまでのパラメータではサーチ個数や処理時間の増加が起こりえます、これまでと同等の処理結果を得たい場合には「粗い精度でのマッチングでの低圧縮スコアしきい値」をいくらか増加させる必要があります。

  • #1050(774) fnFIE_medianMxN()
    • 特定環境でセグメンテーション違反が発生した問題を修正.

  • #1051(775) fnFIE_fpm_feature_window_alloc()
    • 特定の状況で処理速度が低速化する問題を修正.

  • #879(603) fnFIE_niblack_threshold()
    • ARM64bit環境で特定の画像を入力した場合に計算結果がズレる問題を修正.

ドキュメント修正

  • fnFIE_dm_get_minvar
    • 説明の誤字を修正.

  • F_FILTER_KERNEL_T構造体
    • 数式の誤字を修正.
