Prerequisites
=============

git-gq should run on all 
`Linux <https://en.wikipedia.org/wiki/Linux>`_.
distrbutions and probably most
`Unix <https://en.wikipedia.org/wiki/Unix>`_.
variants.

In order to run git-gq you need to have installed these programs/packages:

- git
- `Python <https://www.python.org/>`_ v.3.6 or newer
- colordiff (optional)
- vi or some other text editor
- less (optional)

Standard
`POSIX <https://en.wikipedia.org/wiki/List_of_POSIX_commands#References>`_.
commands like 'cat', 'cp' etc.
